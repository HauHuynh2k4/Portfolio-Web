Ice Quest

2D Snowboarding Game · Unity / C# · PRU212 · FPT University

A 2D snowboarding game built with Unity and C# as a lab assignment for PRU212. Players control a snowboarder racing down icy slopes across 3 progressively harder levels, collecting coins, picking up power-ups, and avoiding obstacles to reach the finish line.

Key Features:

  • 3 Levels — each with unique terrain layout and increasing difficulty
  • Item System — coins, speed boost, shield (blocks one obstacle hit), and helmet power-ups
  • Score System — real-time score tracking with high score persistence via PlayerPrefs
  • Star Rating — 0 to 3 stars awarded per level based on performance
  • Parallax Background — multi-layer scrolling background for depth effect
  • Crash Detection — head collision triggers game over; shield absorbs one hit
  • Audio System — background music and sound effects with AudioMixer
  • Settings — volume control and game settings persistence

Technical highlights:

  • Physics-based player movement using Unity 2D Rigidbody and custom PhysicsMaterial2D
  • Camera follow and background follow scripts for smooth scrolling
  • Item trigger system with individual item behaviors (CoinItem, SpeedItem, HelmetItem, PlayerShield)
  • Global score manager using singleton pattern across scenes
  • Universal Render Pipeline (URP) for 2D lighting and visual effects
  • DOTween integration for UI animations
Unity C# 2D Physics URP DOTween TextMeshPro AudioMixer

Space Explorer

2D Endless Space Shooter · Unity / C# · PRU212 · FPT University

An endless 2D space shooter built with Unity and C# as a lab assignment for PRU212. Players pilot a spaceship through an infinite wave of asteroids, collecting power-ups and chasing the highest score. Difficulty scales dynamically as the score increases, with faster asteroids, curved movement patterns, and small asteroid swarms unlocking over time.

Key Features:

  • Dual Control Modes — play with mouse cursor or keyboard (WASD), switchable from the main menu with preference saved via PlayerPrefs
  • Mana-Based Rapid Fire — hold fire to drain mana for rapid shooting; mana regenerates when released, with a cooldown when fully depleted
  • Progressive Difficulty — asteroid speed, curve intensity, and spawn rate all increase every 10 seconds; hard mode activates at 5000 points
  • Small Asteroid Swarms — a second wave of smaller, faster asteroids unlocks at 500 points
  • Bullet Upgrade System — stacking power-ups evolve the attack: single shot to enhanced shot, then dual shot, then 3-way spread
  • Shield Item — absorbs one hit; flashes as a warning before expiring; stackable to reset duration
  • Fire Rate Boost — speed power-up doubles fire rate for a limited duration
  • Heal Item — restores a portion of the player health bar
  • Top 5 High Score Board — scores saved to JSON; prompts name entry when a new top-5 score is achieved
  • Pause System — press P to pause; includes settings panel with mouse sensitivity control
  • Custom Cursor — custom crosshair cursor in cursor mode
  • Parallax Background — multi-layer scrolling starfield for depth effect
  • Audio System — background music, fire SFX, explosion, heal, shield, and damage sounds via AudioManager singleton

Technical highlights:

  • Singleton pattern used for GameManager, ScoreManager, AudioManager, AsteroidSpawner, and HighScoreManager
  • New Unity Input System for keyboard movement and mouse position reading
  • Coroutine-based timed effects: shield timer with flashing warning, bullet upgrade reset, fire rate reset
  • Score-driven spawn logic: spawn interval decays over time, hard mode flag accelerates decay rate
  • High score persistence using JSON serialization to Application.persistentDataPath
  • Universal Render Pipeline (URP) for 2D lighting and visual polish
  • TextMeshPro for all in-game UI text
Unity C# 2D Physics URP Input System TextMeshPro AudioManager JSON Persistence

1946 Studio

Photobooth Web App · VNR202 · FPT University

A fully browser-based photobooth application themed around Vietnam's National Resistance War of 1946, developed to commemorate the 79th anniversary of the event (19/12/1946 - 19/12/2025). Users can take webcam photos, apply historical frames, and download the final image, all without leaving the browser.

Key Features:

  • Real-time webcam capture with 3-second countdown and flash effect
  • 4 frame layouts: vertical 3-photo, 1 large + 2 small, 2x2 grid, vertical 4-photo strip
  • AI auto-places a soldier hat on detected faces in real-time using MediaPipe FaceMesh
  • Canvas-based photo compositing and high-quality PNG export
  • Vintage UI: aged paper tones, revolutionary red, gold star decorations, classic typography
