Koi Feng Shui Consulting System

Web Application · Team Project

A web application for Koi fish Feng Shui consulting. Users can get personalized Feng Shui recommendations based on their birth year and Koi fish characteristics. The system includes a full admin dashboard for content and user management.

My role: Back-End Developer

  • Designed and implemented RESTful APIs using ASP.NET Core and C#
  • Built the database schema and data access layer with SQL Server and Entity Framework
  • Implemented JWT-based authentication and role-based authorization
  • Developed the core consultation logic for Feng Shui recommendations
  • Collaborated with front-end developers to integrate React UI with the back-end APIs
